# How It Works - MoneyPrinterTurbo Studio

MoneyPrinterTurbo Studio scores topic, script, search terms, voice, subtitles, aspect ratio, and review state before a short-video job moves into production.

## Enter the topic or draft

Start with a product, story, educational topic, or pasted script.

## Score the production inputs

Review script length, aspect ratio, source terms, voice language, subtitle style, and missing assets.

## Resolve review flags

Fix weak hooks, unsupported source assumptions, subtitle mismatch, or batch settings before production.
