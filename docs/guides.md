# Guides - MoneyPrinterTurbo Studio

MoneyPrinterTurbo Studio scores topic, script, search terms, voice, subtitles, aspect ratio, and review state before a short-video job moves into production.

## Prepare a short-video run

Write the topic, audience, CTA, target length, aspect ratio, source policy, voice, and subtitle preference first.

## Review a readiness score

Treat the score as a production checklist: fix missing source terms, unclear script beats, and unsupported media assumptions.

## Use the GitHub repo responsibly

Inspect the upstream repository for rendering code and licenses; use the hosted site for planning, approval, and receipts.

## Pricing and Support

Plans cover short-video runs, batch planning, team review, subtitle and TTS checks, hosted previews, and export receipts.
